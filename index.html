Sobre a empresa

A Mesh Analises é uma empresa voltada à aplicação das técnicas mais robustas e eficientes para solução de análises estruturais via Elementos Finitos.

Sobre nosso escritório

Localizado no interior de JavaScript, nosso escritório opera de forma integrada e totalmente conectada à você e ao que há de melhor na tecnologia de processamento e de análise computacional. Contamos com:

  • Time extremamente capacitado e dedicado

  • Infraestrutura de última geração em análise computacional

  • Licença comercial dos principais softwares de análise, como Ansys, Abaqus e Diana

Tipos de Análises

Análise Estática

As principais análises estáticas das estruturas mecânicas e civil. Análise detalhada com mapa de tensões, deformações, forças internas, diagramas de momento fletor e cisalhante, máximos e mínimos das medidas, caminhos de deformação e de tensão.

Análise Dinâmica

Utilizando processadores de última geração, analisamos suas estruturas a esforços dinâmicos, perfazendo os principais tipos de análises, desde mapas de tensão e deformação no tempo, análises sísmicas, diagrama de respostas no domínio da frequência e no domínio do tempo, animações das estruturas no pós-processamento.

Análise Não-Linear

Fazemos as principais análise de estruturas sujeitas a grandes tensões ou deformações. Análise detalhada do comportamento da estrutura nos estados elástico e plástico, análise de dano, fissuras e trincas, mapa de calor, e diagrama de evolução do dano.

Módulo: Projeto 3

Módulo de Aprendizado

Este projeto se destina à avaliação dos conhecimentos adquiridos nos módulos referentes à tecnologia Bootstrap. Foram dois módulos destinados ao aprendizado do Bootstrap: Fundamentos do Bootstrap e Bootstrap Avançado

Uma ferramenta já pouco utilizada, mas que foi muito importante, é o Bootstrap. Seu paradigma de criar páginas utilizando um sistema pré configurado de grid com vários componentes já estilizados, certamente trouxeram praticidade e agilidade para que desenvolvedores web trouxessem à vida seus projetos.

Por trás dos panos, o Bootstrap utiliza a tecnologia SASS para criar uma arquitetura modular e altamente customizável. Utilizando as várias ferramentas que o SASS oferece, o Bootstrap vem com dezenas de componentes como Modais, Alertas, Abas de Navegação, Tooltips, Formulários, Tabelas, entre outras, já estilizadas e prontas para uso, não requerendo nenhuma estilização ou criação de dinamismo por parte do usuário. Bastava consultar sua documentação e sair utilizando seus componentes.

Hoje em dia, para aplicações mais robustas, principalmente utilizando frameworks bem estabelecidos como o React, Angular ou Vue, o Bootstrap pode até ter perdido um pouco de espaço, principalmente pelo uso extensivo do Tailwind CSS como alternativa, porém, o estudo e aplicação do Bootstrap em módulos de ensino é justificável, principalmente do ponto de vista de sua importância e de seus paradigmas.

O Projeto

Se trata da criação de uma landing page de uma empresa fictícia.

Funcionalidades exigidas e implementadas:

  1. Crie uma outra landing page a partir do layout que construímos durante o módulo - o tema do site fica a seu critério;
  2. Publique o projeto na Vercel.

Funcionalidades adicionais implementadas:

  1. Um footer para informações sobre a implementação;
  2. Máscaras de formato de envio do e-mail e telefone utilizando o plugin Mask do jQuery;
  3. Validação ativa dos campos do formulário, mostrando para o usuário quais campos ele deve preencher e como preencher, utilizando o plugin Validate do jQuery;
  4. Alertas mostrando o envio ou não do formulário, com diferentes classes do Bootstrap para controlar a estilização;

Tecnologias utilizadas

  • Bootstrap v5.3.3
  • HTML5
  • CSS3
  • Vanilla JS (ES6)
  • jQuery v3.7.1
  • Validate, um plugin jQuery v1.19.5
  • jQuery Mask, um plugin jQuery v1.14.16
  • VS Code

Desafios resolvidos e conclusão

Durante a implementação não houve grandes desafios. Talvez o maior fluxo de pensamento e criatividade ficou devido à criação das funções que fazem a adição dos alertas. A estilização foi básica, já que o foco era na utilização do Bootstrap para criação da aplicação. Portanto, mesmo sendo uma tecnologia de entrada nos pacotes de implementação, desenvolvidos para o desenvolvimento web, o Bootstrap ainda se mantém como uma excelente alternativa para desenvolvedores novatos ou simplesmente aventureiros do desenvolvimento web, já que proporciona agilidade e facilidade em lidar com dinâmicas um pouco mais avançadas como Modais e Alertas.
